(gdb) - отладчик
help - подсказка по разделу помощи отладчика/ без параметров - список разделов 
help breakpoints для получения справки по командам, связанным с точками установки.

list - распечатка текста, без индекса(по умолчанию) - следущее 10 строк, распечатываемый файл становится текущим файлом исходного текста
list 1,31 - вывод строчек от 1 до 31, если строк меньше, чем 2 граница, то выведет все строки программы 
list main - вывод начала функции main
list n, Вывести 10 строк, начиная с n. +Enter - ёщё 10 строк и т.д.
list ,m Вывести 10 строк перед m

run начинает исполнение программы до первой точки остановки или возникновения ошибки вроде Floating point

break n  Установить точку останова на строке исходного кода с номером n. Все точки остановки имеют свои номера, определяемые при их установке.
break function Установить точку останова на функции function.
info break Вывести информацию обо всех имеющихся точках остановок

delete a-n / n/   Удалить точки останова с номерами из указанного диапазона. Например, delete 1-5 или delete 4. Если диапазон не задан, удаляются все точки остановок

disable диапазон Отключить точки из заданного диапазона.
enable диапазон Включить точки из заданного диапазона.

finish Продолжить выполнение программы до возврата из текущей функции, напечатать возвращаемое значение.

where Вывести содержимое стека. = backtrace

display Добавить выражение  к списку автоматического отображения.


next - выполнение след строки(залетает в цикл ток если breakpoint
step - след шаг(может покружиться в цикле
